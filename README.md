# 🚀 MahiCV AI – Smart Resume Builder

MahiCV AI is a modern, dynamic CV/resume builder that allows users to create professional resumes with live preview, multiple premium templates, and instant PDF export.

---

## 🔥 Features

* 🧾 Dynamic CV Builder (add/edit sections easily)
* ⚡ Live Preview (real-time updates)
* 🎨 5+ Premium Resume Templates
* 📥 One-click PDF Download
* 🧠 AI-ready (future: summary + resume scoring)
* 🌙 Dark / Light Mode (optional)
* 💾 Auto-save (local storage)
* 📱 Fully Responsive UI

---

## 🛠 Tech Stack

* **Frontend:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **State Management:** React Hooks
* **PDF Generation:** html2canvas + jsPDF
* **Deployment:** Vercel

## 📦 Installation & Setup

```bash
git clone https://github.com/jagdishmaliwad2002/mahicv-ai.git
cd mahicv-ai
npm install
npm run dev
```

App will run on:

```text
http://localhost:3000
```
